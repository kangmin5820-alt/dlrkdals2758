# 필기 입력 기능 사용 가이드

## 📱 기능 소개

PT 세션 중 운동 피드백이나 메모를 태블릿 펜슬로 필기할 수 있는 기능이 추가되었습니다.

## 🎨 UI 구성

### 1. 피드백 입력 화면

```
┌─────────────────────────────────────────┐
│ [텍스트 입력] [✏️ 필기 입력]            │
├─────────────────────────────────────────┤
│                                         │
│  [두께: ▬▬▬▬▬●▬ 5] [색상: ⬜] [지우기] [저장] │
│                                         │
│  ┌───────────────────────────────────┐ │
│  │                                   │ │
│  │      (필기 Canvas 영역)           │ │
│  │                                   │ │
│  │                                   │ │
│  └───────────────────────────────────┘ │
└─────────────────────────────────────────┘
```

## 🖊️ 사용 방법

### 필기 입력

1. **피드백 버튼 클릭** → 피드백 섹션 열기
2. **"✏️ 필기 입력" 버튼 클릭** → 필기 모드로 전환
3. **Canvas에 펜슬이나 손가락으로 필기**
   - 두께 조절: 슬라이더로 1-10 조절
   - 색상 변경: 색상 선택기 사용
   - 지우기: "지우기" 버튼으로 전체 삭제
4. **"저장" 버튼 클릭** → 서버에 업로드 및 저장

### 텍스트 입력

1. **"텍스트 입력" 버튼 클릭** → 텍스트 모드로 전환
2. **텍스트 입력란에 입력**
3. **"저장" 버튼 클릭** → 저장

## 📍 사용 가능한 위치

### 1. 운동 피드백
- 위치: 각 운동 카드의 "피드백" 버튼
- 경로: `/members/[회원ID]/sessions/[세션ID]`

### 2. 세션 메모
- 위치: 세션 페이지 하단의 "오늘의 메모"
- 경로: `/members/[회원ID]/sessions/[세션ID]`

## 🎯 주요 기능

- ✅ 태블릿 터치 지원 (손가락/펜슬)
- ✅ 마우스 지원 (PC에서도 사용 가능)
- ✅ 펜 두께 조절 (1-10)
- ✅ 색상 선택
- ✅ 고해상도 지원 (Retina 디스플레이)
- ✅ 이미지 자동 업로드 및 저장
- ✅ 텍스트/필기 모드 전환

## 💡 팁

- 태블릿에서 사용할 때 가장 좋은 경험을 제공합니다
- 필기 후 반드시 "저장" 버튼을 눌러야 서버에 저장됩니다
- 저장된 필기는 이미지로 표시되며, "수정하기"로 다시 편집할 수 있습니다
